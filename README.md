# OpenMP tutorial for the 2017 MolSSI Summer School

### Compilation

Adjust CXX (C++ compiler) and CXXFLAGS (compilation flags) in the Makefile if
necessary.

To compile individual examples:

	make 00-hello
	make 01-header-and-env
	...

To compile all programs:

	make
